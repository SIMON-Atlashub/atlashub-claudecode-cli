Installation

Guide d'installation de @atlashub/claude-tools Installation guide for @atlashub/claude-tools

Prerequis Prerequisites

  • Node.js >= 18.0.0
  • npm >= 8.0.0
  • Claude Code (CLI d'Anthropic)
# Verifier les versions
node --version  # v18.x.x ou superieur
npm --version   # v8.x.x ou superieur

Installation rapide Quick Install

💡

npm install -g @atlashub/claude-tools && claude-tools install

Installation globale (recommandee) Global Installation (recommended)

L'installation globale installe les commandes dans ~/.claude, les rendant disponibles dans tous vos projets.

Global installation installs commands to ~/.claude, making them available in all your projects.

  1. Installer le package npm
    Install npm package
    npm install -g @atlashub/claude-tools
  2. Activer votre licence
    Activate your license
    claude-tools activate YOUR-LICENSE-KEY
  3. Installer les commandes
    Install commands
    claude-tools install
  4. Verifier l'installation
    Verify installation
    claude-tools status

Installation locale (projet) Local Installation (project)

L'installation locale installe les commandes dans ./.claude du projet courant. Utile pour personnaliser les commandes par projet.

Local installation installs commands to ./.claude in the current project. Useful for customizing commands per project. 

# Dans votre projet
npm install -D @atlashub/claude-tools

# Installer localement
npx claude-tools install --local
⚠
Ajoutez .claude au .gitignore
Add .claude to .gitignore

Si vous installez localement, pensez a ajouter .claude/ a votre .gitignore pour eviter de commiter les fichiers de configuration.

If you install locally, consider adding .claude/ to your .gitignore to avoid committing configuration files.

Options d'installation Installation Options

Option Description Description
--local Installer dans le projet courant (./.claude) Install in current project (./.claude)
--commands-only Installer uniquement les commandes Install commands only
--agents-only Installer uniquement les agents Install agents only
--hooks-only Installer uniquement les hooks Install hooks only
--force Forcer la reecriture des fichiers existants Force overwrite existing files
--no-config Ne pas creer le fichier de configuration Skip config file creation

Desinstallation Uninstallation 

# Desinstaller tous les composants
claude-tools uninstall

# Desinstaller des composants specifiques
claude-tools uninstall --commands-only
claude-tools uninstall --agents-only
claude-tools uninstall --hooks-only

# Garder le fichier de configuration
claude-tools uninstall --keep-config

# Sans confirmation
claude-tools uninstall --yes

Mise a jour Update 

# Mettre a jour le package npm
npm update -g @atlashub/claude-tools

# Mettre a jour les commandes installees
claude-tools update

Resolution de problemes Troubleshooting

⚠
Erreur de permission
Permission error

Si vous obtenez une erreur de permission lors de l'installation globale, utilisez sudo ou configurez npm pour utiliser un repertoire utilisateur.

If you get a permission error during global installation, use sudo or configure npm to use a user directory.

💡
Commandes non reconnues
Commands not recognized

Assurez-vous que le repertoire npm global est dans votre PATH. Verifiez avec npm bin -g.

Make sure the npm global directory is in your PATH. Check with npm bin -g.
