# Checklist Validation BA

## Contexte (4 points)
- [ ] Probleme clairement defini
- [ ] Utilisateurs cibles identifies
- [ ] Valeur business explicite
- [ ] Scope et hors-scope definis

## Modele donnees (6 points)
- [ ] Toutes entites listees
- [ ] Relations definies (FK)
- [ ] Types de donnees specifies
- [ ] Contraintes definies (NOT NULL, UNIQUE)
- [ ] Index suggeres si necessaire
- [ ] Migration EF Core planifiee

## API (7 points)
- [ ] Tous endpoints documentes
- [ ] Methodes HTTP correctes
- [ ] Routes RESTful coherentes
- [ ] DTOs request/response definis
- [ ] Codes HTTP de retour specifies
- [ ] Permissions par endpoint
- [ ] Validations documentees

## Interface utilisateur (6 points)
- [ ] Toutes pages listees
- [ ] URLs definies
- [ ] Wireframes presents
- [ ] Permissions UI definies
- [ ] Flux navigation documente
- [ ] Messages erreur/succes specifies

## Permissions (3 points)
- [ ] Roles identifies
- [ ] Matrice permissions complete
- [ ] Coherence API/UI verifiee

## Technique (3 points)
- [ ] Performance addressee
- [ ] Securite couverte
- [ ] Scalabilite evaluee

---
**Seuil validation** : 85% minimum (25/29)
